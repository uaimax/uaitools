# Deep Research Reports

Esta pasta contém todos os relatórios gerados pelo **Deep Research Agent** do Cursor.

## Estrutura

Os relatórios são salvos automaticamente com o seguinte formato:

```
YYYY-MM-DD-titulo-da-pesquisa.md
```

**Exemplo:**
- `2024-12-19-mercado-saas-b2b-2024.md`
- `2024-12-20-competidores-django-rest-framework.md`
- `2024-12-21-multi-tenancy-django-best-practices.md`

## Formato dos Arquivos

Cada relatório contém:

- **Sumário Executivo**: Insights principais
- **Contexto Histórico**: Origem e evolução do tópico
- **Landscape Atual**: Players principais e estado atual
- **Trends Recentes**: Tendências 2024-2025
- **Métricas Quantitativas**: Dados numéricos relevantes
- **Riscos & Limitações**: Desafios identificados
- **Expert Opinion**: Perspectivas de especialistas
- **Análise Crítica**: Padrões, contradições e gaps
- **Fontes Consultadas**: Bibliografia completa com URLs
- **Confiança da Análise**: Score 1-10 baseado em qualidade das fontes

## Como Usar

Para gerar um novo relatório, use o Deep Research Agent no Cursor:

```
@deep-research [sua pergunta]
```

ou

```
Faça uma pesquisa profunda sobre [tópico]
```

O relatório será salvo automaticamente nesta pasta após a conclusão.

## Histórico

Os relatórios são organizados cronologicamente pela data no nome do arquivo, facilitando a busca e referência futura.
