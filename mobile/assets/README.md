# Assets do Baú Mental Mobile

Este diretório contém os assets visuais do app mobile.

## Assets Necessários

### 1. Ícone do App (`icon.png`)
- **Tamanho**: 1024x1024 pixels
- **Formato**: PNG com transparência
- **Tema**: Baú/tesouro dourado
- **Uso**: Ícone principal do app em todas as plataformas
- **Requisitos**:
  - Fundo transparente ou sólido
  - Design simples e reconhecível em tamanhos pequenos
  - Tema: baú, cofre, tesouro com elementos dourados

### 2. Splash Screen (`splash.png`)
- **Tamanho**: 1284x2778 pixels (iPhone 14 Pro Max)
- **Formato**: PNG
- **Tema**: Baú Mental com fundo escuro (#0D0D0F)
- **Uso**: Tela de inicialização do app
- **Requisitos**:
  - Fundo: #0D0D0F (dark)
  - Elemento central: Baú/tesouro dourado
  - Texto opcional: "Baú Mental"
  - Design minimalista e elegante

### 3. Ícone Adaptativo Android (`adaptive-icon.png`)
- **Tamanho**: 432x432 pixels
- **Formato**: PNG com transparência
- **Tema**: Baú/tesouro dourado
- **Uso**: Ícone adaptativo para Android (versões modernas)
- **Requisitos**:
  - Fundo transparente
  - Elemento central: Baú/tesouro
  - Cores: Dourado (#D4AF37) e complementares
  - Design que funciona bem em diferentes formas (círculo, quadrado, etc.)

### 4. Favicon (`favicon.png` - opcional)
- **Tamanho**: 48x48 pixels (mínimo)
- **Formato**: PNG ou ICO
- **Uso**: Web version do app
- **Requisitos**: Versão simplificada do ícone principal

## Onde Encontrar/Criar Assets

### Opções Gratuitas
1. **Flaticon** (https://www.flaticon.com)
   - Buscar: "treasure chest", "cofre", "baú"
   - Filtrar por licença livre

2. **Unsplash/Pexels** (para referências)
   - Buscar imagens de baús/cofres para inspiração

3. **Canva/Figma**
   - Criar assets customizados usando templates

4. **AI Generators**
   - Midjourney, DALL-E, Stable Diffusion
   - Prompt: "treasure chest icon, gold, minimalist, app icon style"

### Ferramentas Recomendadas
- **Figma**: Design e exportação
- **Adobe Illustrator**: Vetorização
- **GIMP/Photoshop**: Edição de imagens
- **ImageMagick**: Redimensionamento em batch
